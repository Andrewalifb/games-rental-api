Games Rental API

This API provides a platform for users to rent their favorite games. It offers various endpoints to handle user registration, login, balance top-up, and game rental transactions. Users can view their latest and all game rentals, add items to their cart, and proceed to checkout. The API also includes administrative features such as generating business revenue reports, top customer lists, and top games lists. It's designed to provide a seamless game rental experience for users while also providing valuable insights for administrators.

post /users/carts
Add items to cart (addToCart)
This endpoint adds items to the user's cart. It starts a new transaction, generates a new RentalID, calculates the total amount, creates a new payment history, inserts each item into the cart, commits the transaction, and returns a success response.

post /users/carts/checkout/{rental_id}
Checkout cart (checkoutCart)
This endpoint processes the checkout of the user's cart. It retrieves the rental ID from the context, checks if the user's deposit balance is sufficient, updates the payment history, inserts data into the rental transaction table, reduces the user's deposit balance, sends a success payment report to the customer's email, inserts data into the rent maintenance database, removes all rental_id related data from the cart, and returns a success response.

Path parameters

rental_id (required)
Path Parameter —

Return type

ProcessCheckoutResponse

Example data

Content-Type: application/json
{
  "success" : {
    "code" : 0,
    "details" : "details",
    "message" : "message"
  }
}

Produces

  • application/json

Responses
